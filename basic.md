### 基本公式

| 模式          | 公式          |
|---------------|---------------|
| 最右位 1 -> 0 | x and (x - 1) |
| 最右位 0 -> 1 | x or  (x + 1) |
| 尾部 1 -> 0   | x and (x + 1) |
| 尾部 0 -> 1   | x or  (x - 1) |

### 变化公式

| 模式              | 公式             |
|-------------------|------------------|
| 寻找最右位为0的位 | ~x and ( x + 1 ) |
| 寻找最右位为1的位 | ~x or (x - 1)   |
